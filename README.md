# RC8_Driver-Robotiq_FT_Force_Sensor
PacScript driver to communicate between an RC8 controller and the Robotiq FT sensor. 

### Contents

RC8 Driver for reading force/torque data from ATI AI Gamma sensor via TCP/IP.
